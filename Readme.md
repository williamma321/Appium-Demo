# This is a demo using Appium with Python and the BDD (Behave) test framework.

Prerequisite setup to run:

- Xcode 14.3.x
- Python 3.11.x
- Appium 2.1.3
- appium-xcuitest-driver 5.0.0 for Mac
- Selenium 4.12.0
